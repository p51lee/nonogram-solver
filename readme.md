# Nonogram solver

## how to use
```
> sbt
sbt:nonogram-solver> run ./examples/example.txt
```

## input file format
```
width
height
...
informations of each horizontal line
(from top to bottom, from left to right seperated by " ".)
...
informations of each vertical line
(from left to right, from top to bottom seperated by " ".)
...
```
